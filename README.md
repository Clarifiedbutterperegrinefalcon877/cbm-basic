## 📝 What is cbm-basic?

cbm-basic is a small program written in the C language. It works like an old-school computer that used BASIC as its main programming language.  

The interpreter lets you:
- Type simple BASIC commands and see instant results.
- Write and run small programs for learning or fun.
- Explore how early home computers worked.

You don’t need to know how to code to try it. The program provides a window where you enter commands like `PRINT "HELLO"` and it shows the results immediately.

## 🛠️ How to Use cbm-basic

cbm-basic works like a small command prompt. Here are some simple commands to try:

- `PRINT "TEXT"` – Shows the text on the screen.
- `10 PRINT "HELLO"` – Write a line number and a command to create a program.
- `RUN` – Starts running your typed program.
- `LIST` – Shows your entered program.
- `NEW` – Clears the program in memory.
- `HELP` – Displays some basic help commands.

### Writing a Simple Program

1. Type lines starting with numbers, like this:

```
10 PRINT "HELLO"
20 PRINT "THIS IS CBM-BASIC"
30 END
```

2. Type `RUN` and press Enter. The program prints the lines on the screen.

3. Type `LIST` to see your program listed.

4. Type `NEW` to clear the program and start over.

### Saving Your Work

At this time, cbm-basic may not support saving programs to your hard drive. You can copy and paste the text from the window into a text editor like Notepad to save your code manually.
